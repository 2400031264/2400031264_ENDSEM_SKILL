# Student Project Gallery (React + Vite)

## How to run

1. Open a terminal in this folder.
2. Install dependencies:

   npm install

3. Start the dev server:

   npm run dev

4. Open the URL shown in the terminal (usually http://localhost:5173).

You can edit the project data in `src/data/projects.json` or replace the images
in `public/images` with your own.
